核心能力

流速徽章

直接在推文旁显示 views/hour。看到 🚀 小火箭时，说明这条推文正在起爆，值得重点关注。

热点悬浮面板

自动整理当前页面的高流速推文，不用反复刷时间线，也能抓住正在升温的话题。
